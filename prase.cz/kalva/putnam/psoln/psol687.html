29th Putnam 1968


 
 
Problem B1

The random variables X, Y can each take a finite number of integer values. They are not necessarily independent. Express prob( min(X, Y) = k) in terms of p1 = prob( X = k), p2 = prob(Y = k) and p3 = prob( max(X, Y) = k).

 

Solution

Let q1 = prob(X = k, Y = k), q2 = prob(X = k, Y > k), q3 = prob(X > k, Y = k), q4 = prob(X = k, Y < k), q5 = prob(X < k, Y = k). These are all probabilities for disjoint events, so we can add them freely to get: prob( min(X, Y) = k) = q1 + q2 + q3, p1 = q1 + q2 + q4, p2 = q1 + q3 + q5, p3 = q1 + q4 + q5. Hence prob( min(X, Y) = k) = p1 + p2 - p3.
