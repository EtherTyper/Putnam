Problem B2

Find all solutions of the differential equation zz" - 2z'z' = 0 which pass through the point x=1, z=1.

 

Solution

Answer: z = 1/( A(x - 1) + 1).

We have z''/z' = 2 z'/z. Integrating, ln z' = 2 ln z + const, so z' = - A/z2. Integrating again: 1/z = Ax + B. But z(1) = 1, so B = 1 - A.
