32nd IMO 1991 shortlist


 
 
Problem 25

Given n > 1 real numbers 0 ≤ xi ≤ 1, show that for some i < n we have xi(1 - xi+1) ≥ x1(1 - xn)/4.

 

Solution

Suppose first that xn-1 ≥ ½. Then we can take i = n-1, because xi(1 - xi+1) ≥ ½ (1 - xn) > ¼ x1(1 - xn). So we may assume xn-1 < ½.

Now if xi > ½ for some i < n-1, then we can take the largest i for which this holds. So xi+1 ≤ ½. Hence xi(1 - xi+1) > ½ ½ = ¼ ≥ ¼ x1(1 - xn).

If there is no such i, then we must have x2 ≤ ½. Take i = 1. Then xi(1 - xi+1) = x1(1 - x2) ≥ ½ x1 > ¼ x1(1 - xn).
