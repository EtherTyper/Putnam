9th VMO 1970


 
 
Problem A2

Find all positive integers which divide 1890·1930·1970 and are not divisible by 45.

 

Answer

k·2a7b193c197d, where k = 1, 3, 32, 33, 5, 3·5, a = 0, 1, 2, or 3, b = 0 or 1, c = 0 or 1, d = 0 or 1 (192 solutions in all)

 

Solution

1890 = 2·335·7, 1930 = 2·5·193, 1970 = 2·5·197 (and 193 and 197 are prime). So 1890·1930·1970 = 2333537·193·197.
